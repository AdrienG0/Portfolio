# Receptenboek Webapplicatie

Deze webapplicatie stelt gebruikers in staat om recepten toe te voegen, te bewerken, te verwijderen en te bekijken.
## Functionaliteiten

- Recepten toevoegen
- Recepten bewerken
- Recepten verwijderen
- Recepten bekijken

## Doel project
